# 4d-tic

We chose to do a four dimensional tic-tac-toe game. We have thought of a fun way to visualise the 4th dimension and we feel like it would be fun to implement.

Plan for the workflow of the project:
- Game will be stored as a list of lists of lists of lists (4D)
- We need to make an algorithm to filter through the current state of the game after every move to check if a three in a row has been achieved.
- We need to simulate the 3d board as a cube, and the 4th dimension as smaller/ crosses spheres within the translucent bigger ones.
